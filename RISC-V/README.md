# 32-Bit RISC-V Single-Cycle Processor
This project features a functional 32-bit single-cycle CPU based on the RISC-V
Designed and implemented from scratch using SystemVerilog

##
Key Features
- **Full Datapath Integration:** Includes a custom built ALU, Control Unit, 32-register File, and PC logic.
- **Instruction Support:** Implements Arithmetic (ADD, SUB), Logical (AND, OR), Data Transfer (LW, SW), and Conditional Branching (BEQ, BNE).
- **Memory-Mapped Architecture:** Designed with separate Instruction and Data memory subsystems for efficient execution.
- **Verified via Simulation:** Validated using testbenches and waveform analysis to ensure functional correctness of program loops and branching.

## Tools
- SystemVerilog  
- Xilinx Vivado
